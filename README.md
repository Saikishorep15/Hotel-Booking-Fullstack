# 🏨 Hotel Booking System (Full Stack MERN)

A modern full-stack Hotel Booking System built using the MERN stack (MongoDB, Express.js, React, Node.js).

This application allows users to book hotel rooms and view all reservations in real-time. Data is stored securely in MongoDB Atlas (cloud database) and accessed through REST APIs.

---

## 🚀 Features

- 📌 Create new room bookings
- 📋 View all reservations
- ☁️ Cloud database integration (MongoDB Atlas)
- 🔗 RESTful API architecture
- ⚡ Real-time frontend updates
- 🛡 Secure environment variable handling
- 📂 Organized full-stack folder structure

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Axios
- Modern React Hooks (useState, useEffect)

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Dotenv
- CORS

## 📁 Project Structure

hotel-booking-fullstack/
│
├── client/ (React Frontend)

---

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies in both client and server folders
3. Add MongoDB connection string in `server/.env`
4. Start backend: `npm run dev`
5. Start frontend: `npm run dev`
